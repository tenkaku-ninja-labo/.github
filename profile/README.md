# てんかく忍者ラボ README

ようこそ、このリポジトリへ。ここは技術と物語が交差する舞台です。  
私たちは透明性を儀式化し、情報を巻物のように広げ、仲間たちが即興で世界を拡張できるようにしています。

---

## 🥷 脇役・舞台装置
- **あけすけ忍者** : 情報漏洩を笑いに変えるPR忍者。ドキュメントやSNSで活躍。
- **NULL真言** : 空即是色の象徴。技術的透明性を宇宙的エネルギーに変換。
- **CAT KNOWS** : 猫は知っている。観測と遊びの境界を示す存在。

---

## 📜 術：儀式的な技術
- **天空** : RTK-GNSSから天空の理を受信し、精緻な位置を記録する術。
- **撮像** : 高精細（4K、8K）、全方位（360）カメラによって高精細な写像を得る術。
- **脈動** : 位置記録を脈動（Beat）として再構成し、複数のカメラを同期する術。
- **分割** : 動絵巻（動画）を絵巻へと分割し、時刻を刻印する術。
- **検索** : 分割した絵巻物を整理、意味を与え、即座に取り出せるようにする術。
- **検出** : 動絵巻から識別により意味を与える術。
- **追跡** : 物体を認識、付番し複数絵巻への登場を追跡する術。
- **明瞭** : 絵巻物に記された符号、記号を峻別しやすくする術。
- **瞬見** : 即時検出、記録瞬見の術。
- **創造** : 天空、脈動、検出、分割により生成される知を宝箱へと転換する術。
- **分析** : 宝物から知恵を召喚する術。
